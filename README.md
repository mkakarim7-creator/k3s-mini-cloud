# K3s Mini Cloud Cluster

## Description
Ce projet consiste à mettre en place un mini-cloud basé sur Kubernetes
en utilisant K3s (Kubernetes allégé).

## Objectifs
- Installer un cluster K3s
- Déployer des services conteneurisés
- Mettre en place un load balancing interne

## Technologies
- K3s
- Docker
- Ubuntu Server

## Architecture
1 Master Node  
2 Worker Nodes

## Services
- Nginx
- MariaDB
- Grafana
